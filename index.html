BridgeConnect is a full-stack web application designed to foster meaningful connections between different generations. Built with Flask and SQLite on the back-end, it features secure user authentication, full CRUD operations for community posts, and a responsive UI powered by Bootstrap and vanilla JavaScript.

The platform enables seniors and youth to share skills, stories, and experiences through an accessible, intuitive interface that prioritises ease of use for all age groups.
